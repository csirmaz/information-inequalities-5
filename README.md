# information-inequalities-5
Code and dataset for the paper "Information Inequalities for Five Random Variables" by E P Csirmaz and L Csirmaz

# Files

## pts.res

Contains the irreducible staircases for n<=60 in the form of a sequence of 0s and 1s denoting the increase from the previous value.
The optional `+` at the end of the line means that the staircase is kept in the next generation.
